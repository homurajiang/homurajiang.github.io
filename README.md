# 羽毛球对阵生成器

智能匹配，公平对战。纯前端应用，无需后端服务器。

## 功能

- **随机双打**：自动生成双打对阵，保证每人对局数一致
- **混合双打**：支持男女人数不等的场景，自动放松配对约束（不出现男双 vs 女双）
- **单打轮转**：单打循环赛模式
- **实时统计**：比分录入、胜负排名、净胜分统计
- **对局刷新**：在结果页面可重新生成对局（相同参数）
- **历史记录**：自动保存在浏览器 localStorage 中
- **响应式设计**：适配手机和桌面端

## 文件结构


## 部署

### GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 `main` 分支，目录选择 `/ (root)`
4. 保存后等待部署完成

### Vercel

直接导入 GitHub 仓库即可，无需额外配置。

## 匹配算法

- **Card Dealing 算法**：为每个选手生成 k 张"牌"，洗牌后分组，修复组内重复，保证每人恰好 k 局
- **搭档优化**：组内分队时选择重复搭档次数最少的方案
- **混双放松模式**：男女人数不等时，允许"男双 vs 混双"或"混双 vs 女双"配对，但不允许"男双 vs 女双"
